 Netflix SQL Analysis Project

Project Overview
This project analyzes the Netflix dataset using SQL in MySQL Workbench. The objective of this project is to practice SQL queries, data filtering, grouping, sorting, and aggregation techniques to extract meaningful insights from the dataset.

 Tools Used
- MySQL Workbench
- SQL
- GitHub
- Netflix Dataset

Dataset Information
The dataset contains information about Netflix Movies and TV Shows, including:
- Title
- Type
- Country
- Release Year
- Rating
- Director
- Genre

Analysis Performed

 Query 1: Total Records
Counted the total number of records in the dataset.

Query 2: Movies vs TV Shows
Compared the number of Movies and TV Shows available on Netflix.

Query 3: Top Countries
Found the top countries with the highest number of Netflix titles.

Query 4: Content Ratings
Analyzed the distribution of content ratings.

Query 5: Content by Release Year
Examined the number of titles released each year.

Query 6: Recent Movies
Retrieved movies released from 2020 onwards.

Query 7: Longest Movies
Identified the longest movies available on Netflix.

Query 8: Indian Content
Analyzed content produced in India.

Query 9: Top Genres
Found the most common genres in the dataset.

Query 10: Summary Statistics
Generated overall summary statistics from the dataset.
